# ATmark

Obsidian plugin for AT Protocol bookmarking platforms. 

## Supported platforms

- **Semble** (`network.cosmik.*`) - Collections and cards

## Installation

Install via [BRAT](https://github.com/TfTHacker/obsidian42-brat):

1. Install the BRAT plugin from Community Plugins
2. Open BRAT settings
3. Click "Add Beta plugin"
4. Enter: `treethought/obsidian-atmark`
5. Enable the plugin in Community Plugins

## Configuration

1. Open Settings > ATmark
2. Enter your handle or DID
3. Enter your app password (create one at Settings > Privacy and security > App passwords on Bluesky)
